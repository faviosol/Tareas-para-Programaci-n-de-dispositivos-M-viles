# Tareas de Programación de Dispositivos Móviles

Repositorio de ejercicios del curso de Programación
de Dispositivos Móviles - Universidad La Salle Arequipa 2026.

## Autor
Favio André Solórzano Vilca

## Estructura del repositorio

### Práctica 1 - Introducción a Kotlin
- Problema 1: Evaluación de empleados
- Problema 2: Piedra papel tijera
- Problema 3: Adivinar número
- Problema 4: Calculadora

### Practica 2 - Clases, interfaces y más
- Problema 1: Clase Producto para implementar descuento a un pro
- Problema 2: Figuras geométricas con herencia
- Problema 3: Sistema de gestión de biblioteca

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir con IntelliJ IDEA
3. Seleccionar el archivo .kt del problema
4. Ejecutar con el botón Run
