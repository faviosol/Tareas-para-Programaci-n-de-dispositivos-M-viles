# Formulario Mejorado con Jetpack Compose

## Descripción
Aplicación Android desarrollada con Jetpack Compose que implementa un formulario de registro interactivo con validaciones en tiempo real.

## Funcionalidades
- **Campos de texto:** Nombre, Edad y Correo con validaciones.
- **RadioButton:** Selección de género.
- **Slider:** Nivel de experiencia (0-10).
- **Switch:** Indicar si el usuario está activo.
- **Checkbox:** Aceptar términos y condiciones.
- **Botón Enviar:** Habilitado solo cuando todo es válido.
- **Botón Limpiar:** Resetea todos los campos al estado inicial.
- **Resultado:** Visualización en tarjeta con color dinámico según validez.

## Tecnologías
- Kotlin 1.8.10
- Jetpack Compose
- Material3
- Android Studio Giraffe

## Cómo ejecutarlo
1. Clonar el repositorio.
2. Abrir con **Android Studio Giraffe**.
3. Conectar un dispositivo Android o usar el emulador.
4. Ejecutar la aplicación (Run).
